### 문제 요약

1. 과제는 최근에 나온 순서대로
2. 과제 도중 새로운 과제 나오면 기존 과제 중단하고 새로운 과제 진행
3. 새로운 과제 끝나면 이전 과제 하던 부분 계속해서 함

과제 점수 구하기

<br>

**입력**

- 이번 학기 분 수
- N분째에 주어진 과제
    - 1 A T : 과제 만점 A, 과제 해결하는데 T분 걸림
    - 0: 해당 시점에 과제 주어지지 않음

<br>

**출력**

과제 점수

슈도코드

```python
분 수 입력 받기

분 수 만큼 반복:
	새로운 과제 입력 받기
	 if 과제가 주어짐
			과제를 스택에 추가

	 if 과제가 남음
			과제 스택에서 꺼내기
			과제 시간 -= 1
			
			if 과제 끝냄
				과제 점수만큼 더해주기
			if 과제 끝내지 못함
				과제 다시 스택에 삽입			
```
