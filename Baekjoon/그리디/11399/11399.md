### 문제 요약

입력

사람 수 N

각 사람 당 인출 시간 

<br>

출력

인출 시간 총합의 최소값

<br>
예제


총 5명
(Pi = i번 사람이 인출하는데 걸리는 시간)
p1 = 3, p2 = 1, p3 = 4, p4 = 3, p5 =2

<br>


[1,2,3,4,5] 

3+(3+1)+(3+1+4)+(3+1+4+3) + (3+1+4+3+2) 

= 3 +4 + 8 + 11 + 13 = 39

<br>


[2, 5, 1, 4, 3] → 1 2 3 3 4

1 + (1+2) + (1+2+3) + (1+2+3+3) + (1+2+3+3+4)

= 1 + 3 + 6 + 9 + 13 = 32
